#  Fun Learning for Kids! 🎈

A simple, fun, and interactive web application designed to help toddlers and young children learn the alphabet and numbers. The app uses colorful visuals, emojis, and voice synthesis to create an engaging learning experience.

## ✨ Features

*   **Two Learning Modes:** Easily switch between learning ABCs and 123s.
*   **Interactive Cards:** Click on a letter or number to reveal an associated word and a fun emoji.
*   **Text-to-Speech:** Hear the pronunciation of each letter, number, and word, making it great for auditory learners.
*   **Responsive Design:** Built with Bootstrap, it works great on phones, tablets, and desktops.
*   **Colorful & Kid-Friendly UI:** Designed with bright colors and a playful font to keep children engaged.

## 🚀 How to Use

1.  Open the `index.html` file in any modern web browser.
2.  Choose a mode by clicking **"🔤 ABCs"** or **"🔢 123s"**.
3.  Click on any card to see and hear the corresponding word and emoji!

## 🛠️ Technologies Used

*   **HTML5**
*   **CSS3** (with CSS Variables and Grid Layout)
*   **Vanilla JavaScript**
*   **Web Speech API** (for text-to-speech)
*   **Bootstrap 5** (for responsive layout)
