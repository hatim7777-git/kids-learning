#  Fun Learning for Kids! 🎈

A simple, fun, and interactive web application designed to help toddlers and young children learn the alphabet (English and Arabic), numbers, and shapes. The app uses colorful visuals, emojis, and voice synthesis to create an engaging learning experience.

**[🚀 Live Demo](https://hatim7777-git.github.io/kids-learning/)**

## ✨ Features

*   **Multiple Learning Modes:** Switch between learning:
    *   🔤 English ABCs
    *   🔢 English 123s
    *   🔠 Arabic Alphabet
    *   ١٢٣ Arabic Numbers
    *   🎨 Shapes
*   **Interactive Learning Cards:** Click on a letter or number to see and hear examples.
*   **Engaging Quiz Modes:** Test knowledge with dedicated "Alphabet" and "Number" quizzes.
*   **Dynamic Questions:** The quizzes feature multiple question formats (e.g., matching objects, counting emojis) for a fun, replayable experience.
*   **Instant Feedback:** Get immediate audio and visual feedback with a star-based scoring system.
*   **Text-to-Speech:** Hear the pronunciation of each letter, number, and word, making it great for auditory learners.
*   **Custom Audio:** Easily add your own voice recordings for specific words to create a personalized experience.
*   **Responsive Design:** Built with Bootstrap, it works great on phones, tablets, and desktops.

## 🚀 How to Use

1.  Open the `index.html` file in any modern web browser.
2.  Choose a learning mode from the top navigation buttons to explore.
3.  Jump into a quiz mode (**"Alphabet Quiz"** or **"Number Quiz"**) to test your skills!

## 🛠️ Technologies Used

*   **HTML5**
*   **CSS3** (with CSS Variables and Grid Layout)
*   **Vanilla JavaScript**
*   **Web Speech API** (for text-to-speech)
*   **Bootstrap 5** (for responsive layout)
